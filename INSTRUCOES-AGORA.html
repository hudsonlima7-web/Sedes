📋 Instruções Passo a Passo

Siga estas etapas para usar o sistema

1 Limpar Cache do Navegador

Aba aberta: LIMPAR-CACHE.html

Clique no botão "🔄 Limpar Cache Agora"

Aguarde a página recarregar automaticamente (3 segundos)

⚠️ Por que isso é importante?

O navegador guarda versões antigas dos arquivos. Limpar o cache garante que você está usando a versão mais recente do sistema.

2 Criar Aluno Teste (ou Verificar Alunos)

Aba aberta: DIAGNOSTICO.html

Opção A: Se não tem alunos cadastrados:

  • Clique em "➕ Criar Aluno Teste"
  • Aguarde a confirmação
  • A página será recarregada

Opção B: Se já tem alunos:

  • Verifique se seus alunos estão na lista
  • Anote o email e senha de algum aluno

🔑 Credenciais do Aluno Teste:

📧 Email: teste@sedes.com 🔐 Senha: Sedes

3 Fazer Login no Sistema

Aba aberta: index.html

Na tela de login:

  • Digite o email: teste@sedes.com
  • Digite a senha: Sedes
  • Clique em "Entrar"

🔍 Ver Logs Detalhados:

Pressione F12 no teclado para abrir o console e ver todos os logs detalhados do que está acontecendo.

4 Fazer Novo Cadastro (Opcional)

Aba aberta: checkout.html

Se quiser cadastrar um novo aluno:

  • Preencha todos os campos
  • A senha padrão "Sedes" já vem preenchida
  • Clique no botão 👁️ para ver a senha
  • Você pode alterar a senha se quiser
  • Clique em "Finalizar Assinatura"
  • Pressione F12 para ver os logs

✅ Sistema Pronto para Usar!

Todas as abas necessárias foram abertas. Siga os passos acima em ordem.

🐛 Se Algo Não Funcionar:

1. Pressione F12 no navegador

2. Vá na aba "Console"

3. Copie TODOS os logs (Ctrl + A, depois Ctrl + C)

4. Me envie os logs para análise

Com os logs detalhados, consigo identificar EXATAMENTE onde está o problema!
