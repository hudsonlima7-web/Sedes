🔄 Limpar Cache

Forçar atualização do sistema

⚠️ Por que preciso limpar o cache?

O navegador guarda versões antigas dos arquivos para carregar mais rápido. Quando atualizamos o sistema, você precisa limpar o cache para ver as mudanças.

📋 Passo a Passo:

1. Pressione Ctrl + Shift + Delete
2. Marque: "Imagens e arquivos em cache"
3. Clique em "Limpar dados"
4. Pressione Ctrl + Shift + R (recarregar forçado)
🚀 Ir para o Sistema 🔍 Diagnóstico
✅ Cache limpo! A página será recarregada em 3 segundos...
